# Athena
Athena is an AI-powered research assistant for searching, analyzing, and understanding information from documents, web sources, and academic papers. It uses semantic search and advanced RAG to deliver accurate insights with support for document analysis and knowledge graphs.


Project Structure

```
Athena/
├── app.py            # Main application entry point
├── src/              # Core logic and AI modules
├── tests/            # Test and debug files
├── docs/             # Documentation
├── config/           # Configuration and dependencies
├── scripts/          # Setup and utility scripts
├── assets/           # Images and static files
├── .github/          # GitHub workflows and templates
└── LICENSE           # Project license
```
